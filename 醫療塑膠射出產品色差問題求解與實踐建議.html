醫療塑膠射出產品色差問題求解與實踐建議

背景與問題描述

公司做模具塑膠射出成型的醫療產品，生產後於品檢時經常遇到產品顏色差異的問題。基於風險考量（客戶疑慮、防止用錯料、確保原料品質一致性），需要進行把關。

問題：如何在色差問題上把關？能否透過透明度與顏色的規範進行品檢？

想法：利用上一次出貨的產品留樣作為標準件，與新產品在相同光源下同時拍照，比對照片中特定位置的 RGB 或 YMCK 組成。當差異超出規範值時判定 NG。

核心疑問：以上想法是否可行？能否開發軟體工具來鑑別？

專家回覆總結

這是一個非常典型且重要的品質管理（QC）問題。直接回答您的問題：

您的想法在理論上是可行的，且軟體工具絕對開發得出來，但是在「實踐層面」存在許多陷阱。如果不解決環境變數，這個方法會產生極高的誤判率。

一、 您的構想分析與潛在風險

您的做法是：「新舊產品同框拍照」→「比對 RGB/CMYK」→「判斷是否 NG」。這個邏輯看似合理，但有以下幾個技術難點：

1. 標準件的陷阱 (The Drift Problem)

  • 您的做法：用「上一次出貨的產品」當標準。
  • 風險：這是品質管理的大忌，稱為「公差漂移」（Tolerance Drift）。如果第一批產品偏黃 1%，第二批比第一批偏黃 1%，到了第十批可能已經偏黃 10% 卻仍判定合格，因為它只跟第九批比。
  • 修正：必須建立「黃金樣品」（Golden Sample / Master Sample）。所有生產出的產品都必須與最初被客戶承認的那一個標準件（或其複製品）進行比對，而不是跟上一批比。

2. 光源與環境的干擾

  • 問題：RGB 數值不是絕對的物理量，它是「光線」與「物體」交互作用的結果。
  • 風險：雖然您提到「在相同光源下」，但一般室內照明會有頻閃、色溫衰退或是陰影問題。如果是塑膠半透明件，背景顏色（底下墊的紙張顏色）只要有一點不同，RGB 數值就會劇烈跳動。

3. 相機感光元件的「太聰明」

  • 問題：手機或一般相機內部都有強大的 ISP（圖像信號處理）。它們會自動進行白平衡（Auto White Balance）、HDR 補償甚至 AI 美化。
  • 風險：相機會試圖「修正」照片顏色，導致兩次拍攝的數值基準不一致。例如：背景深淺不同，相機曝光值改變，產品的 RGB 就變了。

4. RGB 模型的不足

  • 問題：RGB 是「設備相關」（Device Dependent）的色彩空間。
  • 修正：工業色差比對通常使用 CIELAB (L*a*b*) 色彩空間，並計算 ΔE (Delta E) 值。這是最接近人類視覺感知的色差公式。

二、 如何優化並實踐您的想法？（軟硬體解決方案）

如果您希望自行開發或導入工具來執行此檢查，建議依照以下架構進行，才能達到醫療級的管控：

步驟 1：建立標準化硬體環境（最關鍵的一步）

不要直接拿手機在辦公桌上拍。您需要一個低成本但穩定的物理環境：

  • 標準光源箱 (Standard Light Box)：購買一個小型對色燈箱（通常使用 D65 標準光源），確保每次檢測的光線色溫、亮度完全一致。
  • 固定支架：相機（或手機）必須固定在距離產品相同的高度和角度。
  • 固定背景：使用標準灰卡或純黑/純白無反光材質作為背景。

步驟 2：軟體開發邏輯 (或是尋找現成方案)

開發一個簡單的 Python 程式（利用 OpenCV 函式庫）可以輕鬆達成您的需求。邏輯如下：

  1. 影像校正：在拍攝畫面中放入一張色卡 (Color Checker)。軟體先讀取色卡顏色，對整張照片進行色彩校正，消除相機白平衡誤差。
  2. 色彩轉換：將 RGB 轉換為 L*a*b* 色彩空間。
    • L：亮度
    • a：紅綠值
    • b：黃藍值
  3. 計算色差 (ΔE)：
    軟體抓取產品特定區域的 L*a*b* 值，與資料庫中的「黃金樣品」數值進行公式計算：
    ΔE = √[ (ΔL)2 + (Δa)2 + (Δb)2 ]
  4. 判定標準：
    • ΔE < 1.0：人眼幾乎無法分辨（極優）。
    • ΔE < 2.0：人眼可察覺細微差異（通常是醫療產品的可接受範圍）。
    • ΔE > 3.0：明顯色差（NG）。

步驟 3：針對「透明度」的特殊處理

既然是醫療塑膠射出，透明度（Haze/Transmittance）也是關鍵。

  • 單靠反射光（拍照）很難精準判斷透明度差異。
  • 軟體解法：在產品下方放置一個有高對比圖案（例如黑白棋盤格）的底板。拍照後，軟體分析「透過產品看到的黑白對比度」。如果產品變混濁，黑白對比度會下降。

三、 更推薦的替代方案（降低開發風險）

雖然開發軟體可行，但考量到醫療器材的確效（Validation）成本，購買現成的工業硬體可能是更划算的選擇：

方案 A：手持式色差儀 (Portable Colorimeter) - 高CP值推薦

  • 原理：一個像耳溫槍的小設備，直接貼在塑膠表面「嗶」一聲。
  • 優點：
    1. 直接給出 ΔE 數值，沒有環境光干擾問題。
    2. 數據客觀，可直接匯出報表給客戶（證明 ΔE 在 2.0 以下）。
    3. 價格通常在台幣幾千元到一兩萬元之間，比請工程師開發軟體便宜。
  • 缺點：對於非常透明或形狀極其不規則的產品較難測量。

方案 B：手機 App 搭配專用校色卡

市面上有現成的 App（例如 Pantone Connect 或其他色差分析 App），配合廠商販售的校正色卡，可以利用手機鏡頭做到一定程度的量測。這比自己從頭開發軟體快。

關於「魯爾接頭」形狀要求的補充

問題：色差儀對於產品形狀有甚麼要求，魯爾接頭這樣的產品適合嗎？

這是一個非常切中痛點的問題。標準的手持式色差儀直接測量「魯爾接頭」（Luer Connector）是非常困難且不準確的。

為什麼魯爾接頭不適合直接測量？

  1. 測量口徑（Aperture）不匹配： 一般的色差儀測量孔徑通常為 8mm 或 4mm。魯爾接頭的有效表面積太小，且通常是圓柱狀或圓錐狀（6% taper）。當儀器壓上去時，測量孔無法被產品完全覆蓋，導致「漏光」，數據會完全錯誤。
  2. 曲面與結構干擾： 魯爾接頭有螺紋（Lock）或擋邊，表面不是平的。光線打在曲面上反射角度不一，稍微手抖或位置偏移，讀數就會劇烈跳動。
  3. 透明/半透明材質特性： 醫療用魯爾接頭多為 PC、PP 或 ABS 透明材質。色差儀的光打出去會直接「穿透」產品，測到的其實是產品背後的背景顏色，而不是產品本身。

針對「魯爾接頭」這類小異形件的 4 種解決方案

如果在醫療器材產線上要管控這類產品的色差，業界通常採用以下幾種做法（按推薦程度排序）：

方法一：伴隨生產「標準色板」（Test Plaque）—— 最推薦

這是塑膠射出廠最標準的做法，符合 ASTM 或 ISO 規範。

  • 做法：不直接測量魯爾接頭。而是在模具上（或另外準備一副小模具），使用同一批原料、同樣的參數，射出一個直徑約 20-30mm 的圓形或方形平面色板。
  • 優點：色板是平的，完全適配所有色差儀。測量色板的顏色，就等於測量了該批次產品的顏色（代表性）。
  • 操作：每次開機生產或更換料批時，先打幾片色板進行品檢，合格後才開始量產魯爾接頭。

方法二：測量「料頭/流道」（Runner/Sprue）

  • 做法：如果無法製作色板，可以測量射出後流道廢料上較寬、較平坦的位置。
  • 優點：不需額外模具，且料頭經歷了同樣的熱歷程（Heat History），能反映塑料是否因過熱而黃變。
  • 注意：需確保料頭該位置的平整度能覆蓋測量孔徑。

方法三：使用「微孔徑」色差儀 + 治具

  • 如果您堅持要直接測量成品，必須選購口徑極小（例如 3mm 或 Micro spot）的色差儀。
  • 輔助工具：必須製作一個「定位治具」，確保每次測量角度位置一致。
  • 針對透明件：治具底部必須是標準的「黑腔」或「白板」，確保背景一致。

方法四：透射測量（針對透明度的專業解法）

  • 對於透明醫療器材，通常更關注「黃度指數」（Yellowness Index, YI）和「霧度」（Haze）。
  • 這需要桌上型分光光度儀，並具備透射腔。雖然精確但設備昂貴（台幣 20-40 萬起跳）。

總結實務操作建議

考量到您是為了「把關原料」與「避免用錯料」，方法一（射出標準色板）是成本最低且數據最穩定的方式。

  1. IQC 階段：買一台手持色差儀，直接測量供應商來料的塑膠粒（Pellets）。這需要一個專用的小杯子配件，把料粒裝滿壓實後測量。這樣在生產前就能擋下顏色異常的原料。
  2. IPQC 階段：首件檢查時，保留一塊同批次射出的平整料頭或色板進行比對，確認沒有因為機台溫度過高導致產品發黃。

這樣做既避開了魯爾接頭形狀難測的問題，又能達到您想要的風險控管目的。
